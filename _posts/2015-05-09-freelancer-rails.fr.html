Ruby on Rails is an open-source web framework that’s optimized for programmer happiness and sustainable productivity.

It lets you write beautiful code by favoring convention over configuration, and allows you to get an application up and running very quickly.

Many websites are powered by Ruby on Rails: among others, Some websites built with Ember: GitHub, Shopify, and Soundcloud.

Ruby on Rails was released almost ten years ago, and radically changed how web applications were developped.

Since then, it has gained maturity, stability, and has a strong ecosystem of contributors and users around the world. And new features keep coming, too.

I have been developping with Ruby on Rails for more than four years and have used it in many projects, from open-data applications to e-commerce websites, APIs, micro-services...
